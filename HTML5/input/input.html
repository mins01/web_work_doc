input Element중 <input>의 브라우저별 타입 모양체크

관련 URL : http://www.w3.org/TR/html5/the-input-element.html#the-input-element

위 테이블속 <input>이 브라우저별로 어떻게 보이는가를 테스트 한 것으로 결론을 말하자면 OPERA가 가장 잘 지원하고있다.

  • IE7 : ...논외 HTML5 지원안됨
  • FF : 의외 지원되는게 없다. IE7과 같다.
  • crome : range , search 만 지원됨
  • sfari : 크롬과 똑같다.
  • opera : 가장 많은걸 지원함
    • color 지원안됨
  • 참고
    • search 에 대해서는 text와 같게 동작해도 뭐라할 수 없다.
      Text, Search, URL, Telephone 이들은 애트리뷰트, 이벤트 등이 똑같다.
    • 따로 애트리뷰트를 설정하거나 다른 태그(<datalist> 등)를 같이 사용해야하는 경우도 있지만,
      이 문서의 목적은 단순한 테스트 용이라서 그정도까지 세세히는 다루지 않는다.

결론

  • opera가 가장 잘 지원한다.
  • FF는 HTML5의 <input>의 type을 전혀 지원하지 못한다.
  • 지원이 안되면 그걸 웹개발자가 만들어 줘야한다는 게 되버리는 데...
