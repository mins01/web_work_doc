간단한 sprite 예제

  • 배경 원본이미지

    • sprite.png
    • 너비 100px , 높이 300px
    • 각 너비 50px, 높이 50px씩 이미지가 들어가있음
    • 마지막 왕관 부분은 높이가 잘못되어서 5,6번째 메뉴는 모양이 다르게 보임.

설명

  • 메뉴별 같은 이미지를 레이어로 만든다.(감싸는 li는 overflow:hidden에 postion:relative)
  • 메뉴별 top값을 조정해서 원하는 이미지가 나오도록 한다.
  • :hover로 left값을 -50px로 바꾼다.
  • 끝

장점/단점

  • 장점
    • HTTP로 이미지 전송이 1번만 발생한다.
    • 마우스 오버일 때에도 이미지 전송을 하지 않는다.(304, abort 등일 발생하지 않는다.)
  • 단점
    • 이미지의 크기와 오버시 위치등을 미리 알고있어야한다.
      • 디자이너와 코더의 협의가 필요.
