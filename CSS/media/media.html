요약

  • @media 미디어타입 and (조건) [,미디어타입 and (조건)] { ... }
    • 미디어타입
      • all : 모든 화면에 대해서
      • screen : 일반적 화면
      • projection : 프로젝터
      • tv : 텔레비전
      • ETC : 그외 장치에서 지원이 된다면...
    • 조건
      • width : min와 max 가능
      • height : min와 max 가능
      • device-width : 장치 너비, min와 max 가능
      • device-height : 장치 높이, min와 max 가능
      • orientation : portrait(세로),landscape(가로) (너비와 높이의 비율로 계산됨)
      • aspect-ratio : 비율 (width/height), min와 max 가능
      • device-aspect-ratio : 장치 비율 (width/height), min와 max 가능
      • color : 컬러 비트수, min와 max 가능
        • color 라고 나타내면 모든 컬러 장비에 대한 것
      • color-index : 컬러 수, min와 max 가능
        • color-index 라고 나타내면 모든 컬러 장비에 대한 것
        • 비트가 아니라 수이므로
          min-color-index: 256 로 최소 256색 지원장비표현할 수 있다.
      • monochrome : 흑백 지원 비트수
        • 흑백장비가 아니라면 0
        • 프린터 등에서 사용
          @media print and (monochrome)
      • resolution :dpi, dpcm 등에 대한것
        • 프린터의 dpi 설정에 적용
      • scan : progressive, interlace
        • TV 출력 방식 제한
      • grid : 고정 폰트 기반 출력 방식(tty 터미털, 고정 폰트 장비 등)
