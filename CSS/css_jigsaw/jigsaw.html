jigsaw

  • jigsaw 속 .piece-shape 는 .piece-wrap*(100/70) 의 너비, 높이를 가진다.
  • 돌출부위는 --piece-shape-width 과 --piece-shape-height 의 15% 크기를 가진다.
  • .piece-shape는 .piece-wrap를 벚어날 수 있어서, 순서대로 붙이면 주위의 .piece-shape와 겹친다.
  • 외각 piece는 돌출부의 배경 이미지가 없다!
  • 배경은 .piece-wrap 의 기준으로 위치한다. (.piece-shape의 위치 기준이 아니다!)
