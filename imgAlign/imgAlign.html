  • 비표준
    • baseline,absbottom,absmiddle,texttop
  • absmiddle과 middle 는 조금 다르게 위치한다.(브라우저에 똑같이 보이기도한다. => Opera)
    • absmiddle와 stule의 middle은 같다.(브라우저에 따른 차이 가능)(IE,FF,Opera,Sfari,crome 에서 확인 => 똑같이 보인다.)
  • absbottom = style의 bottom
  • baseline = style의 baseline
  • bottom과 asbbtoom은 조금 차이가 있고 baseline 과는 같다.(IE에서, 브라우저별 차이가 있다.)
  • left와 right는 브라우저마다 표현이 다르게된다.(IE만 다르게 표현됨)
  • 수직 중앙에 이미지가 오는 추천 방법
    • <style>
      img{vertical-align:middle}
      </style>
    • <img alt="이미지" style="vertical-align:middle" src="bg/20br.png" />
