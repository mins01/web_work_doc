css flex 설명
  • css flex : 레이아웃 구조를 잡기위해 사용
  • 링크 , MSDN
  • 아직 브라우저 호환성에 문제가 있음
    -webkit-flex: 1; /* Safari 6.1+ */
    -ms-flex: 1; /* IE 10 */
    flex: 1;
  • css flex : 레이아웃 구조를 잡기위해 사용
  • flex: flex-grow flex-shrink flex-basis|auto|initial|inherit;
    • flex-grow : flex의 다른 item과의 상대적 너비 늘리기 배수. 0 이면 늘어나지 않는다.
    • flex-shrink : flex의 다른 item과의 상대적 너비 줄이기 배수. 0 이면 줄어들지 않는다.
    • flex-basis : 기본너비, %,px 등 사용가능. auto 면 내용의 너비에 맞춰진다.
    • flex:0 0 100px : 100px 고정너비.
    • flex:0 1 100% : 가질 수 있는 최대 너비를 가진다.(ex:제목, 타이틀)
    • flex:0 1 auto : 현재 내용을 기준으로 너비를 가진다.(ex:제목, 타이틀)
    • flex:0 1 100px : 100px 에서 줄어들 수 있다.(ex:코멘트 수)
    • flex:1 1 100px : 100px 를 기준으로 다른 flex와 더불어 여백없이 공간을 차지한다.
  • flex-flow: flex-direction flex-wrap|initial|inherit;
    flex item의 방향과 줄바꿈 정의
    IE는 11부터 지원
    • flex-direction: row|row-reverse|column|column-reverse|initial|inherit;
      flex item의 정렬 방향 정의 (가로,세로,역방향)
    • flex-wrap: nowrap|wrap|wrap-reverse|initial|inherit;
      flex iteml 의 줄바꿈 정의 (nowrap인 경우 수평으로 계속 레이아웃이 유지된다. width가 무시됨 )
  • 기타 관련
    • justify-content : ustify-content: flex-start|flex-end|center|space-between|space-around|initial|inherit;
      flex item의 좌우 정렬
    • align-items : align-items: stretch|center|flex-start|flex-end|baseline|initial|inherit;
      flex item의 상하 정렬
      태그 자체의 값을 바꾸고 싶다면 align-self를 사용하라
    • align-content: stretch|center|flex-start|flex-end|space-between|space-around|initial|inherit;
      flex-wrap에 영향을 미치는 정렬(정렬시의 빈 공간 등)
    • justify-content: center;align-items: center;
      를 사용해서 box의 정중앙에 위치시킬 수 있다. (height가 선언되어있어야함)
css flex 예제
  • flex:0 0 100px // 100px 너비 고정(증가,감소 안됨)
    flex:1 0 100px // 최저 100px 너비, 증가 가능
    flex:2 0 100px // 최저 100px 너비, 증가 가능(이쪽이 더 넓어야함)
  • flex:0 0 100px // 100px 너비 고정(증가,감소 안됨)
    flex:1 0 100px // 최대 100px 너비, 감소 가능
    flex:2 0 100px // 최대 100px 너비, 감소 가능(이쪽이 더 쫍어야함)
  • flex:0 0 100px // 100px 너비 고정(증가,감소 안됨)
    flex:0 0 auto //내용의 길이에 맞춤
  • flex 지정이 없으면 flex:0 1 auto 와 같다.
    flex:0 1 40px; //최대 40px이며, 줄어들 수 있다. 리플 갯수 등을 표시할 때 쓰자
  • flex:0 1 100% //최대 너비를 가진다.
    flex:0 1 40px; //최대 40px이며, 줄어들 수 있다.
